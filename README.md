# Fitness Sports Center

A responsive promotional website for **Fitness Sports Center** — a modern gym brand. Built as part of the Koncepthive Web Development Intern evaluation.

- **Live Demo:** [fitness-sports-center-ebon.vercel.app](https://fitness-sports-center-ebon.vercel.app/)

- **GitHub Repo:** [fitness-sports-center](https://github.com/IT21245060/fitness-sports-center)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React
- Google Fonts (Bebas Neue, Barlow)

## Setup Instructions

### Prerequisites

- Node.js 18+
- npm

### Install and Run

```bash
git clone https://github.com/IT21245060/fitness-sports-center.git
cd fitness-sports-center
npm install
npm run dev
```

Open in browser: `http://localhost:5173`

### Production Build

```bash
npm run build
npm run preview
```
